# 页面智能建议功能 - 快速使用指南

## 功能说明

当您在任意网页上打开 Syncia 侧边栏开始新对话时，系统会自动：
1. 分析当前页面内容
2. 使用 AI 生成 3 条可能对您有帮助的建议
3. 显示在聊天界面顶部
4. 点击任意建议即可开始对话

## 使用步骤

1. **打开侧边栏**
   - 使用快捷键打开 Syncia 侧边栏
   - 或点击浏览器扩展图标

2. **等待建议加载**
   - 首次打开时会显示 3 个加载动画（约 1-3 秒）
   - AI 正在分析页面内容

3. **点击建议**
   - 点击任何一条建议
   - 系统会自动将该建议发送给 AI
   - 开始您的对话

4. **建议消失**
   - 当您发送第一条消息后，建议区域会自动隐藏
   - 切换到新页面时会重新生成建议

## 示例场景

### 📰 新闻文章页面
可能的建议：
- "总结这篇文章的主要观点"
- "分析文章中提到的关键数据"
- "比较文章提到的不同观点"

### 📦 商品页面
可能的建议：
- "比较这个产品的优缺点"
- "寻找类似的替代产品"
- "解释产品的技术规格"

### 💻 技术文档
可能的建议：
- "用简单的语言解释这个概念"
- "提供使用示例"
- "指出可能的注意事项"

## 配置要求

确保已完成以下设置：
- ✅ 已配置 OpenAI API Key（或 Ollama 端点）
- ✅ 已选择 AI 模型
- ✅ 页面内容足够丰富（至少 100 字符）

## 故障排查

**问题：建议不显示**
- 检查是否已配置 API Key 和模型
- 确认页面内容不是空白页面
- 查看浏览器控制台是否有错误信息

**问题：建议加载太慢**
- 可能是网络延迟，请稍等
- 考虑切换到更快的 AI 模型
- 如使用 Ollama，确保本地服务正在运行

**问题：建议不相关**
- 建议基于页面文本内容生成
- 某些页面（如视频、图片为主）可能建议不够准确
- 可以忽略建议，直接输入您的问题

## 技术细节

- 仅在新对话时生成建议（避免打扰进行中的对话）
- 同一页面不会重复生成（已缓存）
- 使用独立的 API 调用，不影响正常聊天
- 建议生成失败时静默处理，不影响其他功能

---

**提示**：如果您不需要建议功能，可以直接忽略，直接在输入框输入您的问题即可。
